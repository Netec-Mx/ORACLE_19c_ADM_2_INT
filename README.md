
# Curso: Base de Datos Oracle: Administración Avanzada Parte 2

Este curso avanzado se centra en la arquitectura multitenant de Oracle y el ajuste del rendimiento de SQL. El enfoque en el diseño, la gestión y la seguridad de los Contenedores de Base de Datos (CDBs) y las Bases de Datos Pluggable (PDBs). Aprenderás a crear, administrar, respaldar y recuperar estas bases de datos de manera eficiente, así como a gestionar la seguridad y el uso de recursos. El curso también se sumerge en técnicas para optimizar el rendimiento de las consultas SQL, abarcando desde la comprensión del optimizador de Oracle hasta el uso de herramientas de ajuste como el Asesor de Ajuste de SQL.
