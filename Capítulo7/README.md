
  
# Capítulo 7. Ajuste de Sentencias SQL
 
## **Práctica 7.1 Ajuste de Sentencias SQL**

En esta práctica aprenderás a identificar, analizar y optimizar sentencias SQL dentro de una PDB. Utilizarás vistas dinámicas para localizar consultas recientes, revisarás su plan de ejecución y generarás un reporte detallado con *DBMS_XPLAN*. Posteriormente, crearás y ejecutarás una tarea del **SQL Tuning Advisor** para obtener recomendaciones del optimizador, como la actualización de estadísticas o el uso de índices. Finalmente, aplicarás las recomendaciones y validarás su impacto.
Esta práctica guía al participante a través del ciclo completo de diagnóstico y mejora de rendimiento en SQL.
